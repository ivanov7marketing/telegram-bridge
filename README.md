# Telegram Bridge API

REST API для работы с персональными аккаунтами Telegram через Pyrogram.

## Возможности

- ✅ Авторизация через QR-код (как в WAHA для WhatsApp)
- ✅ Авторизация по номеру телефона
- ✅ Поддержка 2FA
- ✅ Список диалогов (чатов)
- ✅ Получение сообщений
- ✅ Отправка сообщений
- ✅ **Отправка первого сообщения по номеру телефона** (без предыдущей переписки)
- ✅ Webhook для входящих сообщений
- ✅ Изоляция сессий

## Требования

- Python 3.11+
- Telegram API credentials (api_id, api_hash) от https://my.telegram.org
